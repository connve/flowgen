# Object Store

Flowgen reads, writes, lists, and moves files on S3, GCS, Azure Blob Storage, and local filesystem.

- [Object Store](/docs/flowgen/object-store/object-store) — all operations and configuration.

Credentials vary by cloud provider. See the [Object Store task page](/docs/flowgen/object-store/object-store#credentials) for GCS, S3, and Azure credential formats.
