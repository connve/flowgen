# AI

Flowgen integrates with large language models and tool-calling protocols.

- [AI Completion](/docs/flowgen/ai/completion) — send prompts to an LLM and receive completions.
- [AI Gateway](/docs/flowgen/ai/gateway) — OpenAI-compatible proxy that routes to Vertex AI, Ollama, or other backends.
- [MCP Tools](/docs/flowgen/ai/mcp) — expose flowgen tasks as Model Context Protocol tools.
