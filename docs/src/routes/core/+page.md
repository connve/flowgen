# Core Tasks

Built-in tasks for data transformation, scripting, and flow control.

- [Script (Rhai)](/docs/flowgen/core/script) — run Rhai scripts for custom data transformation.
- [Convert](/docs/flowgen/core/convert) — convert between data formats (JSON, CSV, Arrow, Avro).
- [Iterate](/docs/flowgen/core/iterate) — split an array into individual events.
- [Buffer](/docs/flowgen/core/buffer) — batch events by count or time window.
- [Generate](/docs/flowgen/core/generate) — emit events on a cron schedule.
- [Log](/docs/flowgen/core/log) — log events for debugging.
